# OTTPlus Website Testing

This project includes manual testing of OTTPlus streaming website.

## Scope:
- Functional testing
- UI/UX testing
- Validation testing
- Negative testing

## Modules Covered

- Navigation & Menu  
- Movies, Series, Live TV  
- Login & Subscription Flow  
- UI Validation  
- Responsiveness & Cross-browser Testing  
- Performance & Error Handling  

## Deliverables:
- Test cases
- Bug reports
- Screenshots
- Test summary

## Tools Used:
- Google Sheet
- Browser developer tools
